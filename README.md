To run this project:
- python manage.py runserver

First time install:
- pip3 install django
- pip3 install djangorestframework
- pip3 install django-cors-headers
- pip3 install python-dotenv
- pip3 install psycopg2-binary (to work with PostgreSQL)
- pip3 install openai (use your key from OpenAI platform)
- pip3 install Orange3
  
- move to cloned project
- python3 -m venv django-env
- source django-env/bin/activate
- move to folder orange3_abml_master  , pip3 install -e .
- move to folder orange3_abml_evcrules, pip3 install -e .
